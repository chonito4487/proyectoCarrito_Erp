# ERP Ventas
Proyecto académico de sistema ERP orientado a ventas artesanales (Spring Boot + React + PostgreSQL)

## Estructura
- `/backend` → API REST con Spring Boot
- `/frontend` → Interfaz web con React y Bootstrap

## Objetivo
Simular la gestión completa de compras, ventas y stock en un negocio minorista artesanal paraguayo.
